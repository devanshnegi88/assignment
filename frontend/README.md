# SHL Recommender Frontend

A React + TypeScript frontend for the SHL assessment recommendation backend.

## Setup

```bash
cd frontend
npm install
```

## Run locally

```bash
npm run dev
```

## Production preview

```bash
npm run build
npm run preview
```

## Configuration

The frontend expects the backend API to be available at `VITE_API_BASE_URL`,
which defaults to `http://localhost:8000`.
